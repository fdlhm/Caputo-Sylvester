MATLAB codes accompanying:

Numerical approximation of Caputo-type advection-diffusion equations via Sylvester equations, arXiv:2501.09180, (2025).

If you use this software in your research, please cite the above reference.

The program PDE1.m requires the MATLAB functions herdif, herroots, and poldif, which are part of the MATLAB differentiation suite developed by J. A. C. Weideman and S. C. Reddy. These functions are available at:

https://appliedmaths.sun.ac.za/~weideman/research/differ.html

The programs PDE2.m and PDE3.m require the MATLAB functions cheb.m, available at:

https://people.maths.ox.ac.uk/trefethen/spectral.html
